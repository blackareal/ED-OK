# SWP Math Camp12 — ระบบภารกิจเซลฟี่คณิต

ระบบนี้ทำตามแนว "Selfie Vocab Run" (ค่ายอุดรพิชัยรักษ์พิทยา) แต่เปลี่ยนโจทย์จากคำศัพท์ภาษาอังกฤษเป็นโจทย์คณิตศาสตร์ 30 ข้อ ใช้กับ 8 ทีมสีของค่ายนี้ (แดง ฟ้า เหลือง ส้ม เขียว ม่วง ชมพู น้ำเงิน)

## ⚠️ สถานะ: ชุดภารกิจเป็น "ฉบับร่าง" รอตรวจทาน

ทั้ง 30 ข้อใน `SWP_MathCamp12_SelfieMission.html` และ `SWP_MathCamp12_SelfieBoard.html` เป็นแบบร่างที่ยกตัวอย่างจากโจทย์ที่ครูส่งมาก่อน ยังไม่ได้ตรวจทานความถูกต้อง/ความเหมาะสมกับสถานที่จริง ก่อนใช้งานจริงต้องแก้ไขจุดต่อไปนี้ในไฟล์ (ค้นหาคำว่า `[ครูช่วยยืนยัน]` หรือ `[ครูช่วยระบุ]` ในโค้ด):

- **ภารกิจที่ 4** — ต้องระบุอาคารเรียนจริงที่มีสีตรงกับรหัสสี `#03a9f4`
- **ภารกิจที่ 5** — พิกัด GPS `(17.0817614, 103.8176737)` เป็นพิกัดตัวอย่างที่ครูส่งมา ต้องตรวจสอบว่าตรงกับจุดจริงในผังโรงเรียนหรือไม่ ถ้าไม่ตรงต้องแก้พิกัดในโค้ด (ค้นหา `MISSIONS` ในไฟล์ mission และ `ANSWER_KEY` ในไฟล์ board)
- **ภารกิจที่ 7** — ต้องระบุชื่อครู/บุคลากรจริงที่มีชื่อ/ฉายาความหมายว่า "ความเจริญรุ่งเรืองยิ่งขึ้น"
- **ภารกิจที่ 22** — ใช้จำนวนนักเรียนทั้งหมดในค่าย (ร่างไว้อ้างอิงจาก 240 คนตัวจริง) ควรยืนยันจำนวนจริงอีกครั้งก่อนใช้

นอกจากนี้ ควรพิจารณาว่าโจทย์แต่ละข้อเหมาะกับระยะเวลา/ความปลอดภัยของนักเรียนหรือไม่ (เช่น ภารกิจให้เดินไปยังจุดไกล ควรจำกัดพื้นที่ให้อยู่ในรั้วโรงเรียนเท่านั้น)

แก้ไขเนื้อหาภารกิจได้ที่ค่าคงที่ `MISSIONS` ในไฟล์ `SWP_MathCamp12_SelfieMission.html` และ `ANSWER_KEY` ในไฟล์ `SWP_MathCamp12_SelfieBoard.html` (ต้องแก้ทั้ง 2 ที่ให้ตรงกันถ้าเปลี่ยนเนื้อหาโจทย์)

## ไฟล์ในระบบนี้

| ไฟล์ | ใช้ทำอะไร | ใครเปิด |
|---|---|---|
| `SWP_MathCamp12_SelfieMission.html` | หน้ามือถือ: เลือกทีมสี → ดูภารกิจ 30 ข้อ → ถ่ายรูปอัปโหลดได้ทันทีในหน้านี้ | นักเรียนทุกคน (สแกน QR จากหน้าจอใหญ่) |
| `SWP_MathCamp12_SelfieBoard.html` | หน้าจอใหญ่ 4 แท็บ: แนะนำ+QR / จับเวลา 45 นาที / Dashboard (คะแนน+แกลเลอรีภาพเรียลไทม์) / เฉลย (ครูเท่านั้น) | ครู/ผู้จัด |
| `SWP_MathCamp12_AppsScript.js` | Backend เดียวกับระบบคะแนน (ขยายเพิ่มจากเดิม) — เก็บรูปภารกิจในแท็บ Google Sheet ใหม่ชื่อ `missions` | ครู (paste ใน Apps Script ของ Google Sheet เดิม) |

ไม่ต้องสร้าง Google Sheet ใหม่ ใช้ชีตเดิมของระบบคะแนน (`SWP Math Camp12`) — ระบบจะสร้างแท็บ `missions` ให้อัตโนมัติตอนมีคนอัปโหลดรูปครั้งแรก

## ขั้นตอนติดตั้ง (ทำตามลำดับ)

### 1. อัปเดต Apps Script (สำคัญที่สุด)

ไฟล์ `SWP_MathCamp12_AppsScript.js` ถูกแก้ไขเพิ่มจากเวอร์ชันเดิมที่ใช้กับระบบคะแนน (เพิ่มฟังก์ชันอัปโหลดรูปภารกิจ) เนื่องจากสคริปต์นี้ Deploy ไปแล้วรอบหนึ่ง (มี URL ใช้งานอยู่แล้วสำหรับระบบคะแนน) ต้องอัปเดตโค้ดในที่เดิมแล้ว Deploy ใหม่:

1. เปิด Google Sheet "SWP Math Camp12" (ชีตเดียวกับระบบคะแนน) → เมนู Extensions → Apps Script
2. ลบโค้ดเดิมทั้งหมดในไฟล์ทิ้ง แล้ว paste เนื้อหาทั้งหมดจาก `SWP_MathCamp12_AppsScript.js` (เวอร์ชันล่าสุด) ทับ
3. กด Deploy → **Manage deployments** → กดไอคอนดินสอ (แก้ไข) ที่ deployment เดิม → Version เลือก **New version** → Deploy
4. **ไม่ต้องเปลี่ยน URL** — URL exec เดิมที่ใช้ในระบบคะแนนยังใช้ได้เหมือนเดิม เพราะเป็นสคริปต์เดียวกัน แค่เพิ่มฟังก์ชันใหม่เข้าไป

หลังจากขั้นตอนนี้ ระบบภารกิจจะพร้อมใช้งานทันที เพราะทั้ง `SWP_MathCamp12_SelfieMission.html` และ `SWP_MathCamp12_SelfieBoard.html` ตั้งค่า `SCRIPT_URL` ไว้เป็น URL เดียวกับระบบคะแนนที่ deploy ไว้แล้วอยู่แล้ว ไม่ต้องแก้ไฟล์ HTML ทั้งสองไฟล์นี้เพิ่มเติมในส่วนนี้

### 2. อัปโหลดขึ้น GitHub Pages

1. อัปโหลดไฟล์ `SWP_MathCamp12_SelfieMission.html` และ `SWP_MathCamp12_SelfieBoard.html` ไปที่ repository `ED-OK` บนบัญชี `blackareal` (ผ่าน GitHub Desktop ตามที่ใช้อยู่แล้ว)
2. รอ GitHub Pages build เสร็จ แล้วเปิดทดสอบทั้ง 2 ลิงก์:
   - `https://blackareal.github.io/ED-OK/SWP_MathCamp12_SelfieMission.html` (หน้ามือถือนักเรียน)
   - `https://blackareal.github.io/ED-OK/SWP_MathCamp12_SelfieBoard.html` (หน้าจอใหญ่ครู)
3. ลิงก์ด้านบนถูกใส่ไว้ใน `SWP_MathCamp12_SelfieBoard.html` แล้ว (ตัวแปร `MISSION_PAGE_URL`) — QR โค้ดในแท็บ "แนะนำ" จะชี้ไปที่ลิงก์นี้โดยอัตโนมัติ (สร้างผ่านบริการ QR ออนไลน์ ต้องมีอินเทอร์เน็ตตอนฉายจอ) ถ้าย้ายไฟล์ไปไว้คนละ path จริง ค่อยแก้บรรทัดนี้ในไฟล์อีกที

### 3. ทดสอบก่อนใช้งานจริง

- เปิด `SWP_MathCamp12_SelfieMission.html` ด้วยมือถือจริง เลือกทีมสี ลองอัปโหลดรูปทดสอบ 1 ภารกิจ
- เปิด `SWP_MathCamp12_SelfieBoard.html` แท็บ "Dashboard" เช็คว่ารูปขึ้นและนับคะแนนถูกต้อง
- ลองกดปุ่ม "🔄 เช็คสถานะล่าสุด" ในหน้ามือถือ ดูว่าภารกิจที่อัปโหลดแล้วขึ้นสถานะ "✅" ถูกต้อง
- เปิดแท็บ "เฉลย" ในหน้าจอใหญ่ (ส่วนตัวครูเท่านั้น อย่าฉายให้นักเรียนเห็น) ตรวจทานคำตอบและแก้จุดที่ยังเป็นฉบับร่างตามรายการด้านบน

## คะแนน

ทุกภาพที่อัปโหลดสำเร็จ (ไม่ว่าจะถูกต้องตามโจทย์จริงหรือไม่) = **1 คะแนนเท่ากันทุกข้อ** เต็ม 30 คะแนน — ระดับความยาก (ง่าย/กลาง/ยาก) ใช้เป็นตัวกรองอ้างอิงเท่านั้น เหมือนระบบเดิมของ Selfie Vocab Run เพราะการยืนยันว่ารูปตรงโจทย์จริงหรือไม่ยังต้องให้ครูตรวจด้วยตาจากแท็บ Dashboard/เฉลย (ระบบไม่ได้ตรวจอัตโนมัติ)

## ⚠️ ข้อควรระวังเรื่องความเป็นส่วนตัว

เช่นเดียวกับระบบ Selfie Vocab Run เดิม: หน้าเว็บบน GitHub Pages เป็นสาธารณะเสมอ ต่อให้ repo เป็น private ก็ตาม ใครมีลิงก์ `SWP_MathCamp12_SelfieBoard.html` จะเห็นภาพนักเรียนในแท็บ Dashboard/เฉลยได้ — อย่าโพสต์ลิงก์นี้ในที่สาธารณะ แชร์เฉพาะครู/ทีมงาน และแนะนำให้ปิด/ลบรูปใน Google Drive หลังจบกิจกรรมเพื่อลดความเสี่ยงข้อมูลเด็กหลุดสู่สาธารณะระยะยาว
